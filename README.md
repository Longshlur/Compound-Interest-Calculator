# Compound Interest Calculator

This is a GUI-based calculator built using Python and Tkinter.

## Features
- Calculates compound interest
- User-friendly interface
- Error handling

## Formula
A = P(1 + r/n)^(nt)

## How to Run
1. Install Python
2. Run:
   python compound_interest_gui.py
